Amazon Web Services (AWS)

AWS is a comprehensive, evolving cloud computing platform provided by Amazon. It includes a mixture of infrastructure as a service (IaaS), platform as a service (PaaS), and packaged software as a service (SaaS) offerings.

EC2 (Elastic Compute Cloud)

Amazon EC2 allows users to rent virtual computers on which they can run their own computer applications.

S3 (Simple Storage Service)

Amazon S3 provides scalable object storage for data backup, archiving, and analytics.

RDS (Relational Database Service)

Amazon RDS makes it easy to set up, operate, and scale a relational database in the cloud.
